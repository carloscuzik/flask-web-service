# flask-web-service
Olá, esse repositório é destinado a criação de um simples Web Service utilizando a biblioteca Flask do Python :cow:

## Requirements
- Python (3.6)
- Mysql
- Flask
- Mysql-python

## Run Server

```shell
python server.py
```
